Projects:

  • Magenta is a voicification project developed at Deutsche Telekom to unify voice control of all Telekom products from smart home to TV and telephone. I designed and implemented methods for an NLU model for the German language, 2019 - 2022.
  • BIGGR is a collaborative project between KNIME and ScaDS in Leipzig. We developed a software system for building graph analysis workflows graphically. I implemented GradoopVis and extended the KNIME Analytics platform with new tools. GRADOOP is an open-source system for graph analytics, 2017 - 2019.
  • SIMG-VIZ is a visualization system for entity resolution and clustering. It allows users to analyze precomputed similarity graphs and clustering, and supports ER match tasks with FAMER. It includes features like parallel or serial computation for layouts and sampling for large graphs, 2017 - 2018.
  • PreCol computes graph coloring and preconditioning using Automatic Differentiation, implemented in C++ for efficiency and parallelism. We developed graphical interfaces for Java and MATLAB, 2013 - 2017.
  • MeProRisk II aims to reduce risks in geothermal reservoir exploration by combining skills from different disciplines. I integrated parallel computing into SHEMAT-Suite (SHEMAT), 2012 - 2016.
  • GraphTea is an open-source mathematical platform for graph algorithms and visualizations. I am a developer, designer, and maintainer of GraphTea, which supports both local and online use, 2012 - Now.
  • ChemTea is a set of tools for computing parameters in chemical graph theory using big data approaches. Initially an extension of GraphTea, it evolved into a separate project with C++ code and a Python interface, 2014 - Now.
  • EXPLAIN is a set of interactive learning modules for teaching scientific computing concepts. They are web-based applications written in Python and JavaScript, allowing users to develop and test algorithms online, 2013 - 2016.
  • Selected smaller projects during my master's:
    • Redmine Client: Mobile client for Redmine project management, Humance AG, May 2012 - Nov 2012.
    • Castle Crisis: Android location-based game, Bonn Fraunhofer summer school, July 2012.
    • AggGen: Extracted 3D model of black Carbon using MATLAB/C++, RWTH Aachen, Dec 2011 - Aug 2012.
    • OpenFlipper: C++ library for 3D graphics, developed algorithm for hexagonal mesh, RWTH Aachen, May 2011 - Oct 2011.
